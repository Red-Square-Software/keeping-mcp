    4. **Step order (D-11):**
       a. `actions/checkout@v4`
       b. `actions/setup-node@v4` with `node-version: ${{ matrix.node }}` and `cache: "npm"` (RESEARCH.md Assumption A3 — v4 is the stable choice; v6 exists but v4 has broader doc coverage).

       f. `npx vitest run` (unit tests including token-leak test from Plan 02)
       g. `npm run build` (tsup → dist/bin/keeping-mcp.js)

    5. **Smoke test step (D-13, D-14):** Use `shell: bash` and `env: KEEPING_TOKEN: ""` (Git Bash provides bash on `windows-latest`). The script MUST:
       - Capture stdout: `STDOUT=$(node dist/bin/keeping-mcp.js 2>/tmp/smoke_stderr)`
       - Capture exit code: `EXIT_CODE=$?`
       - Read stderr: `STDERR=$(cat /tmp/smoke_stderr)`
       - Assert exit code ≠ 0 (fail the step with `echo "FAIL: ..." && exit 1` if zero).
       - Assert stdout is empty (`-n "$STDOUT"` → fail).
       - Assert stderr CONTAINS the literal string `[keeping-mcp] Configuration error: KEEPING_TOKEN must not be empty` using `grep -qF "$EXPECTED"`. The `-F` flag is REQUIRED — the `[` characters in the message would otherwise be interpreted as a regex character class and the match would silently fail (RESEARCH.md §ci.yml notes).
       - On success: `echo "Smoke test PASSED: exit_code=$EXIT_CODE, stdout_empty=true, stderr_contains_expected=true"`.
       - Wrap the command in `set +e` / `set -e` so the non-zero exit from the binary does not abort the bash script before assertions run.

    6. **Windows /tmp fallback note:** If the first CI run fails on the `windows-latest` row with a `/tmp` error (Git Bash usually maps `/tmp` to the Windows temp dir, RESEARCH.md Assumption A2), replace `/tmp/smoke_stderr` with `$RUNNER_TEMP/smoke_stderr` and re-run. Do NOT change the assertion logic — only the temp path.

    Do NOT add macOS rows (D-10 explicitly excludes macOS to stay budget-conscious; CONTEXT.md Deferred Ideas defers this).
    Do NOT add a `release.yml` workflow — that's Phase 4 (REL-02..05).
    Do NOT add coverage gating — CONTEXT.md Claude's Discretion defers coverage thresholds.
